This project picks up the basic files from the CenturionDeskmate and replaces the existing main_modular_amber.py 
for a main_xmas_amber.py, as well as the file call on main.py.

Main_xmas_amber.py has a section so it can boot up standalone (then only needing lcd.py) or when called by main.py.

The program presents an animated ASCII sleigh and reindeer with a Christmas message in Portuguese and English in 
a banner configuration
